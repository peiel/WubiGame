[五笔七天速成](https://wubi.yantuz.cn/)

## 使用方法

### 源码运行

运行环境
```
node
```
```
//安装依赖
yarn install
//启动项目
yarn serve
//发布项目
yarn build
```

### 下载发布版本 
到[release](https://github.com/yhf7952/WubiGame/releases)页面下载最新版本，解压后在浏览器中打开index.html，或部署至其他web服务器。
